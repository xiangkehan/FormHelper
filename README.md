# FormHelper

表格识别工具 - 识别 PDF、图片、Word、Excel 文件中的表格内容。

## 技术栈

- **前端**: Vue 3 + Vite + Naive UI
- **桌面框架**: Tauri 2.0
- **后端**: Rust
- **数据库**: SQLite
- **i18n**: 简体中文 / English

## 功能特性

- [ ] 支持 PDF 文件解析
- [ ] 支持图片 OCR 识别
- [ ] 支持 Word 文档表格提取
- [ ] 支持 Excel 文件读取
- [ ] 人员数据管理
- [ ] 表格内容记录与查看
- [ ] 多语言支持

## 开发

### 环境要求

- Node.js 18+
- Rust 1.70+
- pnpm (推荐)

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```
